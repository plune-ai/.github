# Plune

> Proof for AI products. Open-source eval toolkit for teams shipping with LLMs.

**Status:** Pre-alpha. Coming soon.

We're building Plune — a CLI-first evaluation toolkit that helps QA 
and engineering teams regression-test LLM outputs, run AI-as-judge 
scoring, and gate AI releases in CI.

- [ ] v0.1 — CLI with exact-match / contains / json-schema / llm-judge assertions
